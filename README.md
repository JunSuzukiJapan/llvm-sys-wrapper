# llvm-sys-wrapper

Shallow wrapper for llvm-sys.

# Installation

include the following in your Cargo.toml file:

```
[dependencies]
llvm-sys-wrapper="0.1.0"
```

And then import the library using:

```
#[macro_use]
extern crate llvm_sys_wrapper;
```
